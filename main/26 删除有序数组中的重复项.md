# [26 删除有序数组中的重复项](https://leetcode.cn/problems/remove-duplicates-from-sorted-array/)

## 暴力解法

```c++
int removeDuplicates(vector<int>& nums) {
    int size = 0;
    for(auto iter = nums.begin(); iter != nums.end() - 1;){
        if(*iter == *(iter + 1)){
            nums.erase(iter + 1);
        }else{
            ++iter;
            ++size;
        }
    }
    return ++size;
}
```

## 快慢指针

```c++
int removeDuplicates(vector<int>& nums) {
    int fast = 1;
    int slow = 0;
    for(; fast < nums.size(); ++fast){
        if(nums[fast] != nums[slow]){
            ++slow;
            nums[slow] = nums[fast];
        }
    }
    return slow + 1;
}
```
